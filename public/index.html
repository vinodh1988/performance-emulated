MongoDB Performance All-Round

Generic direct MongoDB dashboard for logs, profiler, slow queries, bottlenecks, replica health, and synthetic workload checks.

Tutorial

MongoDB Connection

The app connects directly with the MongoDB driver. No SSH, no mongosh, no VM-specific commands.

Synthetic Load

Create a realistic lab workload through the MongoDB driver. The script creates customers, orders, events, indexes, and enough data for profiler and explain-plan practice.

MongoDB Logs

Uses getLog from MongoDB admin commands. No OS log file access required.

Performance

Compares query plans, documents examined, keys examined, and execution time.

Bottlenecks

Checks serverStatus, WiredTiger cache, locks, connections, and database stats.

Profiler

Enables a short profiler window, reads system.profile, and disables profiling.

Result

Idle 
Enter MongoDB connection details, then choose a check.
